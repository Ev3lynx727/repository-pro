# repository-pro

Professional GitHub repository creation skill for AI agents. Creates well-structured, production-ready repositories with industry best practices.

## Description

AI-powered skill for creating professional GitHub repositories with:
- Proper structure for any tech stack (Node.js, Python, Go, etc.)
- Essential files (README, LICENSE, .gitignore, CONTRIBUTING.md)
- CI/CD workflows (GitHub Actions)
- Issue and PR templates
- GitHub topics and descriptions for discoverability
- Security configuration (Dependabot, branch protection)

## Installation

```bash
npx skills add Ev3lynx727/repository-pro
```

Or from URL:

```bash
npx skills add https://github.com/Ev3lynx727/repository-pro
```

## Usage

This skill triggers when you need to:
- Create a new GitHub repository
- Set up project structure
- Add essential repository files
- Configure CI/CD workflows
- Set up contribution guidelines
- Add issue/PR templates
- Configure GitHub topics and description
- Apply tech stack templates (React, Next.js, Django, FastAPI, Go, etc.)

## Features

### Core Features
- README.md templates (multiple styles)
- LICENSE file generation
- .gitignore templates
- CONTRIBUTING.md guidelines
- Issue templates (Bug Report, Feature Request)
- PR template
- CI/CD workflow templates

### Tech Stack Support
- Node.js / Express / TypeScript
- React / Next.js / Vue
- Python / FastAPI / Django / Flask
- Go / Gin / Echo
- Rust
- Docker / Kubernetes
- MERN, T3 Stack, JAMstack

### CI/CD Templates
- Node.js with coverage
- Python with linting
- Go with cross-compilation
- Docker build & push
- Multi-stage builds
- Security scanning

### Metadata
- GitHub topics recommendations
- Description templates
- Repository name guidelines

## Quick Start

When user asks to create a repository:

1. Identify the tech stack from description
2. Select appropriate templates
3. Create repository with all essential files
4. Apply proper GitHub topics
